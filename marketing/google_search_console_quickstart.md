# Google Search Console Quickstart for Times Table Animals

## 1) Confirm your production URL
Current assumed URL in this repo:
- https://dalloliogm.github.io/Times-Table-Animals/

If you use a custom domain, update these files before submission:
- robots.txt
- sitemap.xml

## 2) Add property in Search Console
- Open Google Search Console
- Add property using URL-prefix
- Enter your exact production URL

## 3) Verify ownership
Use one of the supported methods:
- HTML file upload
- HTML tag in home page
- DNS record (recommended for custom domain)

## 4) Submit sitemap
In Search Console > Sitemaps, submit:
- https://dalloliogm.github.io/Times-Table-Animals/sitemap.xml

## 5) Request indexing
In URL Inspection, test and request indexing for:
- Home page URL

## 6) Monitor for 2-4 weeks
Track:
- Impressions
- Average position for query: times table animals
- Click-through rate
- Coverage issues (Excluded/Crawled not indexed)

## 7) Improve based on query data
- Expand on-page text for real search queries
- Add internal links to key sections
- Refine title/description for better CTR
- Add pages for parent and teacher intent if needed
