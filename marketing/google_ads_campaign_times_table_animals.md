# Google Ads Campaign Plan: Times Table Animals

## Objective
Drive high-intent traffic for searches related to times table learning with animals, and optimize for first meaningful in-game engagement.

## Campaign Setup
- Campaign type: Search
- Campaign name: Search | Times Table Animals | Core
- Bidding (start): Maximize Clicks with CPC cap, then move to Maximize Conversions after reliable conversion volume
- Networks: Google Search Network only (disable Display in this campaign)
- Locations: Start with your target country (single market first)
- Languages: English
- Ad schedule: 6am-10pm local time to start
- Devices: All (adjust after 7-14 days)

## Recommended Budget
- Daily budget: $20-$30 for first 14 days
- Evaluation window: 14 days minimum before major structural changes

## Conversion Events (Priority)
1. Primary: First level completion
2. Secondary: Start game click
3. Secondary: Session > 60 seconds

## Campaign Structure (Ad Groups)

### 1) Brand-Core
Intent: Users searching specifically for your product/phrase.

Keywords:
- [times table animals]
- "times table animals"
- [times-table-animals]
- "times table animals game"

### 2) Times Tables Game
Intent: Users searching for game-based times-table practice.

Keywords:
- "times tables game"
- [times tables game for kids]
- "multiplication game kids"
- "times tables practice game"
- [fun times tables]

### 3) Animal Math Game
Intent: Users explicitly wanting animal-themed math learning.

Keywords:
- "animal math game"
- "math game with animals"
- [multiplication animals game]
- "kids animal learning game"

### 4) Parent-Teacher Intent
Intent: Adult decision-makers.

Keywords:
- "times tables activities for kids"
- "best multiplication game for kids"
- "classroom multiplication games"
- [math game for primary school]

## Negative Keywords (Campaign-Level)
- free worksheets
- worksheet
- printable
- pdf
- algebra
- calculus
- university
- casino
- betting
- answers key

### Descriptions
- Turn multiplication into a fun animal adventure kids enjoy and complete daily.
- Practice times tables with playful habitats, rewards, and clear progress.
- Designed for children, useful for parents, and classroom-friendly for teachers.
- Help kids build speed and confidence in multiplication through game-based learning.

## Landing Page Mapping
- Brand-Core -> homepage with exact phrase "Times Table Animals" above the fold
- Times Tables Game -> section emphasizing multiplication drills and progression
- Animal Math Game -> section highlighting habitats/animal theme
- Parent-Teacher Intent -> section with educational value and outcomes

## URL Tracking Template
Use this final URL suffix at campaign level:

utm_source=google&utm_medium=cpc&utm_campaign=times_table_animals_search&utm_term={keyword}&utm_content={creative}

## Launch Checklist
1. Confirm conversion tags fire for all 3 events.
2. Add at least 1 RSA per ad group (2 preferred).
3. Add sitelinks: Start Game, How It Works, Habitats, For Parents/Teachers.
4. Add callouts: Kid-Friendly, Curriculum Support, Game-Based Learning, No Signup Friction.
5. Add structured snippets: Topics (Multiplication, Times Tables, Mental Math).

## Optimization Plan (Week 1-3)

### Every 3 days
- Pause keywords with high spend and zero meaningful conversions.
- Add search term negatives from irrelevant queries.
- Shift budget toward best-performing ad groups.

### Weekly
- Test 3-5 new headlines and 2 new descriptions.
- Review device performance and adjust bids by device.
- Split out top-performing queries into exact-match ad groups.

## KPI Targets (Starter Benchmarks)
- CTR: > 5% on branded group, > 3% on non-branded groups
- Conversion rate (primary): improve week-over-week
- Cost per first level completion: trending downward by week 2-3

## What to Avoid
- Do not mix broad Display traffic into this Search campaign.
- Do not overuse broad match in week 1.
- Do not optimize too early (< 7 days data unless severe waste).
